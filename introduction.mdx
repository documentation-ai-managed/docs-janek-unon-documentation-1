## Overview

Janek Unon is a modern documentation platform that helps you create, organize, and publish project documentation with ease. You can build structured knowledge bases, collaborate with teams, and deliver polished docs to users. Designed for developers, product managers, and technical writers, it supports MDX for rich, interactive content.

<Callout kind="info">
  Janek Unon uses MDX components to create engaging pages with tabs, steps, and more.
</Callout>

## How to Get Started

Follow these steps to set up your Janek Unon documentation site.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at https://janek-unon.com with your email.
  </Step>
  <Step title="Initialize Project" icon="package">
    Create a new documentation space and connect your Git repository.
  </Step>
  <Step title="Add Content" icon="file-text">
    Write your first page using MDX syntax.
  </Step>
  <Step title="Publish" icon="globe">
    Deploy live and share your docs URL.
  </Step>
</Steps>

Here is a quick example of adding a code snippet to your docs:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Fetch documentation data
  const response = await fetch('https://api.janek-unon.com/docs');
  const docs = await response.json();
  console.log(docs);
  ```
  ```python
  # Fetch documentation data
  import requests
  response = requests.get('https://api.janek-unon.com/docs')
  docs = response.json()
  print(docs)
  ```
</CodeGroup>
