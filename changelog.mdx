## Upgrade Process

Follow these steps to upgrade Janek Unon smoothly.

<Steps>
  <Step title="Backup Data" icon="database">
    Export your current documentation space.

    ```bash
    janek-unon export --space=default --output=backup.json
    ```
  </Step>
  <Step title="Install New Version" icon="download">
    Use your package manager to update.
  </Step>
  <Step title="Migrate Config" icon="settings">
    Convert config files.

    <CodeGroup tabs="npm,yarn">
      ```bash
      npm install janek-unon@latest
      ```
      ```bash
      yarn add janek-unon@latest
      ```
    </CodeGroup>
  </Step>
  <Step title="Restart Service" icon="refresh-cw">
    Restart and verify.
  </Step>
</Steps>

<Tabs>
  <Tab title="Docker" icon="docker">
    Pull the latest image and redeploy.

    ```bash
    docker pull janekunon/docs:latest
    docker-compose up -d
    ```
  </Tab>
  <Tab title="NPM" icon="package">
    Update globally.

    ```bash
    npm update -g janek-unon
    janek-unon migrate-config
    ```
  </Tab>
</Tabs>
